# 小算包（miniCalculator）

> 本项目使用小程序框架mpvue扩展的微信小程序，内容包括进制数值转换，几何面积计算，生辰八字测算等计算工具。

## 功能

### 数值转换

- 角度弧度相互转换
- 进制转换
- 长度、面积、体积、质量、温度、功/能/热转换

### 计算工具（均附有canvas图形展示）

- 圆面积、周长，半径计算
- 矩形周长、面积、边长计算
- 三角形角度、边长、面积、周长计算
- 多边形面积、周长计算
- 生辰八字测算，农历、五行、方位、星座、生肖测算
